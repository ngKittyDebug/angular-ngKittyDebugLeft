#Sprint 1: Project Setup & Component Basics - 2026-05-18

**Что произошло**:

Сегодня доделал компонент страницы 404 (not-found). Данный компонент, как и все остальные, `standalone`. Используется `ChangeDetectionStrategy.OnPush`.
Сам компонент не сложный и значительных трудностей по работе с Angular не возникло.

Основная трудность возникла с созданием анимированного изображения, далее буду писать gif-изображение. Придумать идею и сгенерировать несколько видео-роликов получилось довольно таки быстро, примерно за один - два часа. Но тимейт из команды предложил сделать gif-изображение c прозрачным фоном. Для этого пришлось изучать работу некоторых видео-редакторов. В итоге получилось у меня убрать фон, с помощью редактора Camtasia Studio, а для конвертации в gif и окончательного преобразования фона в прозрачный онлайн сервис https://ezgif.com. К сожалению сам видео-редактор не мог экспортировать изображение с прозрачным фоном, такую возможность убрали. Поэтому пришлось воспользоваться сервисом. После его использования оставались артефакты, которые портили впечатление. Мой перфекционизм не позволил использовать такое gif-изображение, поэтому на данном этапе решил использовать gif-изображение без прозрачного фона.

Тимейт, процессе мое разработки компонента, сгенерировал вариант изображения с использованием svg-анимации. Огромное спасибо за это. Но это вариант мне совсем понравился внешне и поэтому решил попробовать реализовать свой вариант. Тимейт также подсказал хороший AI-сервис для генерации изображений и видео. Огромное ему спасибо!
